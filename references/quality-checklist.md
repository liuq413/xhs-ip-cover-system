# Quality Checklist

## 1. Thumbnail Check

Pass only if:

- The main title is readable at small size.
- The title has enough contrast.
- The user can understand the topic in 3 seconds.
- The character does not cover the title.

Fix if:

- The title is too small.
- Decorative stickers compete with the title.
- The background is too dark or noisy.
- The text has garbled characters.

## 2. Character Consistency Check

Pass only if:

- The same character identity is visible.
- Fixed recognition points are preserved.
- The outfit/accessory matches the selected mode.
- The expression fits the content.

Fix if:

- The character looks like a random mascot.
- The face changes too much.
- Clothing contradicts the selected style variant.
- The character becomes too childish for the topic.

## 3. Information Density Check

Pass only if:

- There are 3-5 meaningful information blocks.
- Each card/sticker explains the topic.
- There is only one main visual idea.
- The cover does not try to teach the whole article.

Fix if:

- There are too many cards.
- Text is too small.
- Stickers are purely decorative.
- Multiple diagrams compete for attention.

## 4. Brand System Check

Pass only if:

- The cover can belong to a series.
- The style matches the creator's positioning.
- The character adds recognition.
- The palette can be repeated.

Fix if:

- The style is attractive but not reusable.
- The character is not needed.
- The cover could belong to any creator.
- The palette changes too randomly.

## 5. Platform Fit Check

Pass only if:

- It feels like a Xiaohongshu knowledge cover.
- The hook is clear.
- The design is clean enough for mobile feed.
- It invites a tap without looking like spam.

Fix if:

- It looks like a poster instead of a feed cover.
- It is too subtle to compete in feed.
- It is too loud for the creator's positioning.
