# Examples

## Example 1: Beginner Workflow Cover

Brief:

```text
Content type: Beginner explainer
Main title: 从0搭AI流程
Topic: AI workflow setup
Reader promise: beginners can understand the basic steps
Reusable framework: five-step workflow
Character mode: Friendly mascot mode
Style variant: Cream Companion
Mood: safe, beginner-friendly
Aspect ratio: 3:4
```

Recommended direction:

- Cream bear hat.
- Character holding laptop.
- Five cards: demand, standard, flow, sample, template.
- White/yellow/cyan/cream palette.

## Example 2: Tool Stack Cover

Brief:

```text
Content type: Tool tutorial
Main title: 我的AI工具流
Topic: tool combination for design productivity
Reader promise: readers can copy a useful tool stack
Character mode: Friendly mascot or professional mode
Style variant: Black Efficiency
Mood: efficient, clear, tool-focused
Aspect ratio: 3:4
```

Recommended direction:

- Black bear hat or black jacket.
- Tool cards and node links.
- Avoid real logos unless user provides them.
- Strong title with list-style cards.
