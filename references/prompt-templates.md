# Prompt Templates

## Universal Cover Prompt

```text
Create a Xiaohongshu cover image, vertical 3:4, polished commercial illustration style.

Topic: [topic]
The main title must be large, clear, and readable: "[main title]"

Main character:
[character prompt block]

Character action:
[action]

Layout:
Top: [small tag]
Upper-middle: oversized main title
Middle: [cards/stickers/diagram elements]
Lower-right or lower-middle: personal IP character
Bottom: [positioning tag or benefit tag]

Visual style:
[style variant description]

Information elements:
[element 1]
[element 2]
[element 3]

Constraints:
The title must be readable in a small mobile feed.
No watermark.
No unrelated props.
No extra people.
No garbled text.
No long English paragraph.
No real brand logo unless provided by the user.
```

## Professional Mode Character Block

```text
Main character is [character name], a 3D chibi personal IP character.
Fixed traits: [fixed face traits].
Outfit: [professional outfit].
Mood: professional, clear, trustworthy, and approachable.
Role: [creator's domain] mentor explaining a practical workflow.
Keep the character recognizable across covers.
```

## Friendly Mascot Character Block

```text
Main character is [character name], a 3D chibi personal IP character.
Fixed traits: [fixed face traits].
Outfit/accessory: [friendly mascot outfit/accessory].
Mood: warm, approachable, low-pressure, and companion-like.
Role: a friendly guide for [creator's domain].
Do not turn the character into a generic toy mascot.
Keep the professional content context visible through props and information cards.
```

## Example: Workflow Explainer

```text
Create a Xiaohongshu cover image, vertical 3:4, polished commercial illustration style.

Topic: AI workflow beginner explainer.
The main title must be large, clear, and readable: "从0搭AI流程"

Main character:
Main character is Q版阿清, a 3D chibi personal IP character.
Fixed traits: black short hair visible under the hat, large eyes, young Asian male, friendly smile.
Outfit/accessory: cream bear-ear hat and light khaki utility hoodie.
Mood: warm, approachable, low-pressure, and companion-like.
Role: a friendly guide for AI workflow and design productivity.
Do not turn the character into a generic toy mascot.

Character action:
Holding a small laptop and pointing to a five-step workflow card.

Layout:
Top: "新手入门"
Upper-middle: oversized main title
Middle: five workflow cards
Lower-right: Q版阿清
Bottom: "AI+设计提效"

Visual style:
Cream Companion style: warm white and cream palette, soft cards, yellow and cyan accents, clean sticker collage.

Information elements:
Cards labeled "拆需求", "定标准", "搭流程", "跑样本", "沉模板".
Arrows, sticky notes, checklists, node links.

Constraints:
The title must be readable in a small mobile feed.
No watermark.
No unrelated props.
No extra people.
No garbled text.
No long English paragraph.
No real brand logo unless provided by the user.
```
