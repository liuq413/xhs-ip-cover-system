# Style Variants

## Variant 1: Cream Companion

Use for beginner explainers, basic tutorials, and soft educational content.

Visual traits:

- Cream or light khaki bear hat.
- White, cream, warm yellow, and cyan palette.
- Soft sticker cards.
- Friendly expression.
- Small laptop, prompt card, sticky notes.

Avoid:

- Too much toy-like styling.
- Excessive pastel colors.
- Props unrelated to learning.

Prompt phrase:

```text
Cream Companion style: warm white and cream palette, soft bear-hat friendly mascot, gentle beginner-friendly knowledge cover, sticky notes, checklist cards, small laptop, approachable but still professional.
```

## Variant 2: Black Efficiency

Use for tool lists, AI tool workflows, efficiency systems, and prompt/tool content.

Visual traits:

- Black or dark gray bear hat.
- White, black, cyan-blue, and yellow palette.
- Tool cards, node graphs, abstract UI blocks.
- Smart and slightly cooler expression.

Avoid:

- Full dark background that hurts thumbnail clarity.
- Real tool logos unless provided by the user.
- Overly cyberpunk effects.

Prompt phrase:

```text
Black Efficiency style: black bear-hat friendly mascot, clean white background, black/cyan/yellow contrast, tool cards, node links, prompt cards, efficient AI workflow feeling, readable knowledge cover.
```

## Variant 3: Utility Fieldwork

Use for case studies, implementation, project workflows, and "from idea to execution" content.

Visual traits:

- Khaki or olive utility bear hat.
- Utility vest or fieldwork jacket.
- Project board, process map, sticky notes, progress bars.
- Practical, executable, grounded.

Avoid:

- Outdoor adventure feeling that disconnects from AI/design work.
- Too many rugged details.
- Military-looking tone.

Prompt phrase:

```text
Utility Fieldwork style: khaki/olive bear-hat mascot, utility jacket, practical project-board layout, process cards, sticky notes, workflow arrows, execution-focused but friendly.
```

## Variant 4: Street Sprint

Use for hot topics, challenges, fast-start tutorials, and high-hook covers.

Visual traits:

- Sport or streetwear bear hat.
- Orange, electric blue, yellow, white, black outline.
- Speed lines, launch stickers, action posture.
- Energetic and clickable.

Avoid:

- Using it for every cover.
- Too many colors.
- Making the character look like a children's sports toy.

Prompt phrase:

```text
Street Sprint style: sporty streetwear bear-hat mascot, orange/electric-blue/yellow contrast, speed lines, launch-card stickers, high-energy Xiaohongshu hook cover.
```

## Selection Matrix

| Content need | Use |
|---|---|
| Gentle entry | Cream Companion |
| Tools and efficiency | Black Efficiency |
| Project execution | Utility Fieldwork |
| Hot topic or quick action | Street Sprint |

## Long-Term Use

Do not use all variants equally. Pick:

- One primary style.
- One secondary style.
- One occasional high-impact style.

For an AI/design workflow creator, a strong default set is:

- Primary: Utility Fieldwork
- Secondary: Black Efficiency
- Beginner-friendly: Cream Companion
- Occasional hook: Street Sprint
